## ec2 launcher script

### How to use 

```sh
 -  First create an .env file using the .env.example present
    cp .env.example .env

 - Fill in the values in the .env

 - In the ec2.py fill the maxcount of instances

 - Install dependencies
  pip3 install -r requirements.txt

 - Run the script 
  python3 ec2.py

  - Visit your aws dashboard you instances should be creating

```

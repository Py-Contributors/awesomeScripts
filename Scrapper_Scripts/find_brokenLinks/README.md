## A Quick Script to Find Any Broken Links on Websites

**get all the broken links from website input via CLI** 

## Modules

- requests==2.18.4
- bs4==0.0.1
- urllib3==1.22

### How to run

```pip install -r requirements.txt ```

```python brokenlinksfinder.py <website>```

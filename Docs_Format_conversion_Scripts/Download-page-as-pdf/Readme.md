Download a page as a PDF .

Required Modules :
pyppdf
  pip3 install pyppdf
pyppyteer
  pip3 install pyppeteer
Examples of use :
Download a page:
   python download-page-as-pdf.py -l 'www.pudim.com.br'
Download a page and give a pdf name:
   python download-page-as-pdf.py -l 'http://www.pudim.com.br' -n 'pudim.pdf'
